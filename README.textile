h1. Algorithmic recruitment

Spider the social graph of a geographic subset of GitHub users and rank them by network-analysis metrics.

h2. Requirements

@JRuby@
@gem install bossman@

h2. Running the ranker

# Get a Yahoo API key (https://developer.apps.yahoo.com/wsregapp/)
# Copy config.yml.example to config.yml and add in your key
# ruby graph.rb 'New York'
